An LLM-Driven Agent for Cross-Language Standardization of Laboratory Test Names

Abstract

Laboratory testing plays a critical role in clinical decision-making and forms a substantial part of electronic medical records (EMRs). Standardizing laboratory test names is essential for achieving semantic interoperability across healthcare systems. The Logical Observation Identifiers Names and Codes (LOINC) system provides a global standard for laboratory data, yet mapping local test names to LOINC codes remains challenging, especially across different languages. We propose an LLM-driven agent framework to address these challenges. The framework divides the mapping process into three components: Tool Component for translation and retrieval, Reasoning Component for semantic inference, and Validation Component for verification. Evaluation on two real-world datasets—the Guangdong Provincial People's Hospital (GDPH) dataset (Chinese) and the MIMIC-IV dataset (English)—demonstrated substantial performance gains. Compared to rule-based and similarity-matching methods, the agent achieved up to a 30–40% relative improvement in mapping accuracy. These results highlight the agent’s potential to enable scalable, cross-lingual standardization of laboratory test names and promote global healthcare data interoperability.
